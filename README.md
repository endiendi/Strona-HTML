# Strona-HTML

Krótki opis tego, czym jest ten projekt i jaki jest jego cel.

## Technologie

Lista technologii użytych do budowy projektu:
* HTML5
* CSS3
* JavaScript

## Jak uruchomić

Aby wyświetlić stronę, wystarczy otworzyć plik `index.html` w dowolnej przeglądarce internetowej.

1. Sklonuj repozytorium na swój lokalny dysk.
2. Otwórz plik `index.html` w przeglądarce.
